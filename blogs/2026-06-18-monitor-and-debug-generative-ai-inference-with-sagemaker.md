This post introduces SageMaker's detailed observability metrics and Insights dashboard for monitoring generative AI inference at scale, emitting over 100 metrics covering GPU health, token-level latency, KV cache utilization, traffic distribution, and cold-start diagnostics. The dashboard offers Performance, Capacity, and Reliability tabs, supports automatic enablement on new endpoints and opt-in on existing ones, connects to tools like Grafana via PromQL, and follows CloudWatch OpenTelemetry ingestion pricing at $0.50 per GB.
