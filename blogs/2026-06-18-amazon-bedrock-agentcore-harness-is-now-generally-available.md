Amazon Bedrock AgentCore harness reached general availability as a managed abstraction for running production agents with minimal configuration, using two API calls (CreateHarness and InvokeHarness) without custom orchestration code or container building. It supports multiple model providers (Bedrock, OpenAI, Gemini, LiteLLM) with mid-session provider switching that preserves context, bundles tools like AgentCore Gateway, remote MCP servers, browser sandbox, code interpreter, and inline functions, and streams every invocation in real time with automatic CloudWatch tracing.
