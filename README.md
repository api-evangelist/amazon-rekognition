# Amazon Rekognition

Amazon Rekognition is a cloud-based computer vision service that makes it easy to add image and video analysis to your applications, providing capabilities such as object and scene detection, facial analysis, face comparison, celebrity recognition, and text detection using deep learning technology.

## APIs

- **Amazon Rekognition Documentation** - Official AWS documentation for Amazon Rekognition, providing comprehensive guides, API references, and tutorials for implementing computer vision capabilities.
- **Amazon Rekognition OpenAPI** - The OpenAPI definition for the Amazon Rekognition API, describing all available operations for image and video analysis including label detection, facial analysis, and text recognition.
- **Amazon Rekognition API Reference** - Complete API reference documentation for Amazon Rekognition with detailed descriptions of all operations, parameters, and data types.
